# Data Studio

# Introdução ao Data Studio

O **Data Studio** é o módulo dedicado à exploração, análise e visualização dos dados operacionais dos dispositivos monitorados.  
Ele permite transformar grandes volumes de dados brutos em informações visuais, facilitando a compreensão do comportamento dos ativos ao longo do tempo.

O módulo é utilizado tanto para análises rápidas quanto para investigações mais detalhadas, servindo de apoio para tomadas de decisão técnicas e operacionais.

---

# Timeseries

A funcionalidade de **Timeseries** permite a visualização dos dados em formato de série temporal.  
Por meio dessa visualização, é possível acompanhar a evolução de variáveis ao longo do tempo, identificar padrões, tendências e eventuais anomalias.

Normalmente, os gráficos de série temporal permitem:

- Seleção de dispositivos e variáveis
- Definição de intervalo de tempo
- Comparação entre diferentes sinais
- Análise visual de variações e comportamentos atípicos

Essa abordagem facilita o entendimento do contexto operacional e complementa análises realizadas em outros módulos da plataforma.
